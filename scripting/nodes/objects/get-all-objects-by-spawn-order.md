# Get All Objects By Spawn Order
![](../../../.gitbook/assets/get-all-objects-by-spawn-order.png)
## Description
Returns a list of all objects with the matching Spawn Order

## Node Type
Nodes fall into two basic categories: Data and Execution. This node supplies Data for an Execution node.

## Inputs
| Input | Type | Required | Description |
|------------------|------------------|----------|--------------------------------------------------------------|
| Spawn Order | Spawn Order | Yes | Spawn Order to use for get all objects. |

## Outputs
| Output | Type | Description |
|------------------|------------------|--------------------------------------------------------------|
| New Object List | Object List | List of all valid objects with the Spawn Order. |
